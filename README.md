# Конвертер валют с историей



Простое мобильное приложение для конвертации валют. Пользователь вводит сумму и пару валют, получает актуальный курс и может просматривать историю своих запросов.



---



## Основные функции



- **Ввод данных**: пользователь вводит сумму и выбирает исходную и целевую валюту.

- **Конвертация**: приложение запрашивает актуальный курс у публичного API и рассчитывает результат.

- **Отображение результата**: на главном экране отображается итоговая сумма в целевой валюте.

- **История запросов**: все использованные пары валют сохраняются в список на отдельном экране.

- **Управление историей**: пользователь может удалять записи из истории.

- **Быстрый доступ**: при нажатии на запись в истории, приложение возвращает пользователя на главный экран с уже заполненными полями выбранной пары валют.



---



## Структура экранов



### 1. Главный экран: "КОНВЕРТЕР ВАЛЮТ"



- **Поля ввода**:

&nbsp; - Поле для ввода **суммы** (например, `100`).

&nbsp; - Поле для выбора **исходной валюты** (например, `USD`).

&nbsp; - Стрелка `→`.

&nbsp; - Поле для выбора **целевой валюты** (например, `RUB`).

- **Кнопка**: `КОНВЕРТИРОВАТЬ`.

- **Область вывода**: прямоугольник, в котором отображается результат конвертации (например, `100 USD = 9234.50 RUB`).



### 2. Экран "ИСТОРИЯ"



- **Заголовок**: `ИСТОРИЯ`.

- **Список записей**: каждая запись — это строка в формате `валюта a → валюта b`.

- **Кнопка удаления**: иконка корзины справа от каждой записи.

- **Навигационная панель внизу**: две кнопки `Главная` и `История` для переключения между экранами.



---



## Навигация



- **Главный экран → История**: переход по нажатию на кнопку `История` в нижней панели.

- **История → Главный экран**:

&nbsp; - По нажатию на кнопку `Главная` в нижней панели.

&nbsp; - По нажатию на любую запись в списке истории — приложение автоматически возвращается на главный экран и заполняет поля исходной и целевой валюты данными из выбранной записи.

- **Удаление из истории**: нажатие на иконку корзины удаляет соответствующую запись из списка.
